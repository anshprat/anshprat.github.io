--- layout: post title: Bzr checkout error date: 2010-01-04 15:15:23.000000000 +01:00 categories: [] tags: - blog - tech status: publish type: post published: true meta: _edit_last: '1' author: --- I was trying to clone the [[keystone package][https://code.launchpad.net/ubuntu/+source/keystone]] using the given bzr command. `bzr branch lp:ubuntu/trusty/keystone` However, I got the following error: bzr: ERROR: Revision {package-import@ubuntu.com-20131016120940-8ite3zq710ktlx3c} not present in "Graph(StackedParentsProvider(bzrlib.repository._LazyListJoin(([CachingParentsProvider(CallableToParentsProviderAdapter())], []))))". A bit of googling lead me to [[https://bugs.launchpad.net/bzr/+bug/888615]] which has a workaround.. `bzr -Olaunchpad.packaging_verbosity=off branch lp:ubuntu/trusty/keystone` The above command worked flawlessly!
